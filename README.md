# Overview
JednorÄ™ki bandyta (One armed Bandit) is a game that was created as a programming project for the subject Programowaine III. The game is made in C# language using Godot game engine. It is a game for 2 players and can be played over the network. The goal of the game is to defeat the opponent. Each player has a gun that can shoot up to 3 bullets, until it needs to reload. The players are skeletons and the game theme is casino style.
